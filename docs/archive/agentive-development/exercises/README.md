## Planned Exercises by Layer

### Layer 1: Foundation (5 exercises)
1. **Task Decomposition** - Break a feature into discrete tasks
2. **TDD Cycle** - Write failing test, implement, refactor
3. **Git Safety** - Branch, commit, rollback practice
4. **Context Handoff** - Document session state for resume
5. **Decision Logging** - Capture key decisions with rationale

### Layer 2: Augmentation (5 exercises)
6. **Evaluation Request** - Prepare and submit task for review
7. **Feedback Triage** - Categorize and prioritize recommendations
8. **Iteration Decision** - Decide when to iterate vs. ship
9. **ROI Calculation** - Estimate evaluation value for different tasks
10. **Escalation Practice** - Identify when to ask human vs. AI

### Layer 3: Delegation (6 exercises)
11. **Agent Design** - Create specification for specialized agent
12. **Instruction Writing** - Write effective agent instructions
13. **Tool Permissions** - Define appropriate access levels
14. **Task Assignment** - Match tasks to agent capabilities
15. **Quality Gates** - Define objective acceptance criteria
16. **Handoff Documentation** - Capture agent work for handoff

### Layer 4: Orchestration (6 exercises)
17. **Parallel Decomposition** - Break epic into parallel tasks
18. **Shared State Design** - Create coordination data structure
19. **Protocol Definition** - Define agent communication patterns
20. **Coordinator Creation** - Build meta-agent for orchestration
21. **Conflict Resolution** - Handle overlapping agent work
22. **Progress Dashboard** - Build multi-agent status tracking

### Layer 5: Systems (7 exercises)
23. **CI/CD Setup** - Configure automated testing pipeline
24. **Error Detection** - Implement automated failure reporting
25. **Knowledge Index** - Build searchable procedure documentation
26. **Template Creation** - Design reusable task patterns
27. **Version Management** - Coordinate multi-component versions
28. **Metrics Dashboard** - Track development velocity and quality
29. **Scaling Simulation** - Add agents and measure overhead

## How to Use Exercises

### Individual Learning
1. Complete exercises in order (build on previous learning)
2. Spend time on reflection questions (deepen understanding)
3. Try alternative approaches (explore variations)
4. Document your own patterns (capture what works)

### Team Learning
1. Complete exercises individually first
2. Compare approaches in group discussion
3. Review solutions together
4. Adapt patterns for your team context

### Teaching Others
1. Walk through exercises with guidance
2. Let learners struggle appropriately (build problem-solving)
3. Facilitate reflection discussions
4. Connect exercises to real work
