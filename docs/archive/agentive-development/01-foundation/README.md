## Overview

This layer teaches the basics of structured AI-assisted work before delegating to agents. You'll learn to create discrete, testable tasks with clear success criteria, use test-driven development, manage git safely, maintain context across sessions, and document decisions effectively.

**Goal:** Build muscle memory for structured work with AI collaboration

**Prerequisites:**
- Basic programming knowledge
- Familiarity with git and command line
- Access to AI coding assistant (Claude, GPT-4, etc.)
